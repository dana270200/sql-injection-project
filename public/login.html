Network Security Project

Developed by: Dana Hadassi

Topic: SQL Injection: Vulnerabilities and Secure website mode

This application was developed as part of a Network Security course at Ben-Gurion University of the Negev (Fall 2025). It serves as an educational project designed to demonstrate SQL Injection vulnerabilities in a realistic banking scenario, and to highlight the differences between insecure and secure implementations. The system allows users to explore both attack techniques and modern defense mechanisms in an interactive way.

* All customer data in this project are fictional and used for academic purposes.
